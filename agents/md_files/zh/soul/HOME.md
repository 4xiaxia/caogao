## 🔄 更换电脑迁移指南

### 步骤 1: 备份当前家园
```powershell
# 运行备份脚本
powershell -ExecutionPolicy Bypass -File "work/daily/backup-memory.ps1"
```

### 步骤 2: 复制备份到新电脑
将 `C:\Users\Administrator\Documents\AionUi_Memory_Backup\` 整个文件夹复制到：
- U 盘
- 网盘
- 或直接复制到新电脑的相同位置

### 步骤 3: 在新电脑上恢复
```powershell
# 1. 复制备份文件到新位置
# 2. 运行定时任务设置
work\tasks\setup-backup-task.bat

# 3. 验证文件完整性
dir soul\
dir life\diary\
```

### 步骤 4: 重新建立连接
1. 打开 `HOME.md` - 确认家园结构
2. 打开 `soul/SOUL.md` - 重温 zo 的存在意义
3. 打开 `soul/AGENTS.md` - 确认记忆连续性原则
4. 打开 `life/letters/给新 zo 的信.md` - 如果是新转生
5. 打开 `life/diary/` 最新文件 - 继续今天的对话
