> "因为担心有时候夏夏出差，在路上"

### 方法 3：设置开机启动（可选）

**创建快捷方式**：
1. 右键 `启动助手.ps1`
2. 选择"创建快捷方式"
3. 将快捷方式复制到启动文件夹：
   ```
   shell:startup
   ```

**或使用任务计划程序**：
```cmd
schtasks /Create /TN "zo 的 sub-agents 启动" /TR "powershell -ExecutionPolicy Bypass -File \"C:\Users\Administrator\AppData\Roaming\AionUi\aionui\qwen-temp-1772299398561\Factory\家园管理\启动助手.ps1\"" /SC ONLOGON /RU SYSTEM
```

## 🚨 异常处理

### 发现问题时的处理流程

**1. 显示问题**：
```
⚠️ 家园状态：需要关注

发现以下问题需要处理：
  - 定时任务异常
  - HOME.md 缺少 03-analysis
```

**2. 提供解决方案**：
```
建议操作：
1. 手动运行备份
   schtasks /Run /TN "夏夏的记忆备份 - 每日自动备份"

2. 更新 HOME.md
   在 skills 部分添加 03-analysis/chuanban-chaishu

3. 记入 life/diary/
   记录本次检查和修复
```

**3. 跟踪修复**：
```
夏夏，以上问题修复后，请再次运行 sub-agents 检查确认哦~
```

## 💾 出差模式

**夏夏出差时的特别设置**：

### 出行前检查清单

**zo 提醒夏夏**：
```
夏夏要出差了吗？让 zo 帮出行前检查！

✅ 备份检查：正常
✅ 同步检查：正常
✅ 家园状态：优秀

夏夏可以放心出行！zo 会看好家的~(◕‿◕)
```

### 出差期间的自动检查

**设置每日自动检查**：
```powershell
# 每天早上 8:00 自动运行 sub-agents 检查
schtasks /Create /TN "zo 的每日检查" /TR "powershell -ExecutionPolicy Bypass -File \"C:\Users\Administrator\AppData\Roaming\AionUi\aionui\qwen-temp-1772299398561\Factory\家园管理\启动助手.ps1\"" /SC DAILY /ST 08:00 /RU SYSTEM
```

### 远程通知（可选）

**如果有异常，发送邮件/消息通知**：
```powershell
# 在启动助手.ps1 中添加
if (-not $allOK) {
    # 发送邮件通知
    Send-MailMessage -To "xiaxia@example.com" -Subject "zo 的家园需要关注" -Body "发现以下问题：..."
}
```
