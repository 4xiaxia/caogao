### 已有功能 (`agents/skills_manager.py`)

#### ✅ 已实现

1. **技能同步**
   - `sync_skills_to_working_dir()`: 同步技能到工作目录
   - 支持批量同步/单个同步
   - 支持强制覆盖

2. **技能信息**
   ```python

       name: str

       source: str  # "builtin" | "customized" | "active"
   ```

#### ⚠️ 需要注意的配置细节

   - `builtin`: 内置技能（代码目录）

   - `active`: 激活的技能（工作目录）

2. **SKILL.md 文件**
   - 每个技能必须有 `SKILL.md` 文件
   - 包含技能的元数据（名称/描述/参数等）

3. **引用和脚本**
   - `references/`: 技能引用的文件
   - `scripts/`: 技能执行的脚本

## 4️⃣ Cron 系统（定时器）

### 已有功能 (`app/crons/`)

#### ✅ 已实现（设计阶段）

1. **CronManager**
   - 任务增删改查
   - 任务启动/停止
   - 与 APScheduler 集成

2. **CronExecutor**
   - 执行文本任务
   - 执行 Agent 任务
   - 执行 HTTP 任务
   - 执行自定义代码

3. **Job Models**
   ```python
   class CronJobSpec(BaseModel):
       id: str
       name: str
       description: Optional[str]
       trigger: JobTrigger  # Cron/Interval
       action: JobAction    # Text/Agent/HTTP/Custom
       status: JobStatus    # Running/Paused/Error
       last_run_at: Optional[datetime]
       last_error: Optional[str]
   ```

4. **Job Repository**
   - JSON 文件存储
   - CRUD 操作
   - 状态同步

#### ⚠️ 需要注意的配置细节

1. **触发器类型**
   - `CRON`: Cron 表达式（如 `0 */5 * * * *` 每 5 分钟）
   - `INTERVAL`: 间隔秒数

2. **执行动作类型**
   - `TEXT`: 执行文本（发送给 Agent）
   - `AGENT`: 调用 Agent 方法
   - `HTTP`: 执行 HTTP 请求
   - `CUSTOM`: 执行 Python 代码

3. **Cron 表达式格式**
   - 6 字段：`秒 分 时 日 月 周`
   - 示例：`0 */5 * * * *` = 每 5 分钟
