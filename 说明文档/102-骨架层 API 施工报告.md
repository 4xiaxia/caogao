## 💖 3. Soul 保护 API (`/agent/soul`)

### 端点清单

| 方法 | 路径 | 功能 | 返回数据 |
|------|------|------|---------|
| GET | `/soul/files` | Soul 文件列表 | 文件列表 + 保护标记 |
| GET | `/soul/memory/{name}/backups` | 备份历史 | 备份列表 |
| POST | `/soul/memory/{name}/rollback` | 回滚到备份 | 回滚结果 |
| POST | `/soul/memory/{name}/request-modify` | 请求修改 Soul | 确认 + 备份路径 |
| GET | `/soul/moments` | 珍贵瞬间列表 | 瞬间列表 |
| GET | `/soul/protection/status` | 保护状态 | 保护机制状态 |

### 核心数据结构

```json
// GET /agent/soul/files
{
  "files": [
    {
      "filename": "SOUL.md",
      "path": "...",
      "exists": true,
      "protected": true,
      "requires_backup": true,
      "last_modified": "2026-03-02T10:00:00Z",
      "size": 1024
    }
  ],
  "total": 11
}

// GET /agent/soul/memory/SOUL.md/backups
{
  "filename": "SOUL.md",
  "backups": [
    {
      "backup_path": "~/.copaw/backups/soul/SOUL_20260302_120000.md",
      "timestamp": "2026-03-02T12:00:00Z",
      "operation": "modify",
      "size": 1024
    }
  ],
  "total": 5
}
```

### 保护机制

- ✅ 删除保护 - Soul 文件禁止删除
- ✅ 修改备份 - 修改前自动备份
- ✅ 版本回滚 - 支持回滚到任意备份版本
- ✅ 二次确认 - 修改需要用户确认

### 施工说明

- ✅ 备份目录：`~/.copaw/backups/soul/`
- ✅ 备份命名：`{filename}_{timestamp}.md`
- ✅ 回滚前自动创建当前版本备份
- ⏳ 待实现修改日志记录
