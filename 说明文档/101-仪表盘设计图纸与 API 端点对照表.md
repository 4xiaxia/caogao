| **首页智能推荐** | 需要 AI 逻辑，实现复杂 | 简化为静态推荐位，后期增强 |
| **成就进度** | 技能等级系统未定义 | 与 Skills 模块合并，避免重复 |
| **工作归档 vs Storage** | 功能重叠 | 统一为 Storage 模块 |

### 2. API 合并建议

| 原设计 | 优化后 | 说明 |
|--------|--------|------|
| `GET /home/overview` + `GET /home/factory-preview` | `GET /home/dashboard` | 合并为一个 Dashboard API |
| `GET /home/projects` + `GET /home/skills` | 包含在 `/home/dashboard` | 减少请求次数 |
| `GET /factory/stats` + `GET /factory/agents` | `GET /factory/dashboard` | 工厂仪表盘 |

### 3. 数据结构优化

```python
# 优化前：多个独立 API
GET /home/overview      → {pending_tasks, expiring_tasks, ...}
GET /home/projects      → {projects: [...]}
GET /home/skills        → {level, total_xp, ...}

# 优化后：统一 Dashboard API
GET /home/dashboard
Response: {
  "overview": {...},
  "projects": [...],
  "skills": {...},
  "factory_preview": {...},
  "recent_activities": [...]
}
```
