## 2️⃣ 频道列表（埋点标注）

```markdown
## 📱 频道列表

⚡ 获取频道列表
├─ ID: channel-list-get-001
├─ API: GET /communication/channels
├─ 数据：💾 channels (数组)
├─ 事件：onMount + 📡 WebSocket 推送
└─ 备注：
    ├─ 📦 数据来源：所有可用渠道
    └─ 💾 字段：id, name, type, status, last_message, unread_count

💾 频道条目
├─ ID: channel-entry-001
├─ 数据：
│  ├─ 💾 channel.id
│  ├─ 💾 channel.name
│  ├─ 💾 channel.type (private/system/agent/external)
│  ├─ 💾 channel.status (online/offline/busy)
│  ├─ 💾 channel.last_message
│  └─ 💾 channel.unread_count
├─ 事件：
│  └─ 🔌 onClick: selectChannel
└─ 备注：
    ├─ 🎨 status=online → 绿色圆点
    ├─ 🎨 status=offline → 灰色圆点
    └─ 🎨 unread>0 → 显示未读数

🔄 实时更新
├─ ID: channel-list-realtime-001
├─ API: WebSocket /ws/channels
├─ 数据：💾 channelUpdates (推送)
├─ 事件：
│  ├─ 📡 onMessage: updateChannel
│  └─ 📡 onError: fallbackToPolling
└─ 备注：
    ├─ 🔄 备用方案：每 10 秒轮询
    └─ 🎨 新消息 → 动画高亮

🔌 切换频道
├─ ID: channel-switch-001
├─ 事件：
│  ├─ 🔌 onClick: switchChannel
│  └─ ⚡ 获取频道消息：GET /communication/channels/{id}/messages
├─ 数据：💾 selectedChannelId
└─ 备注：
    ├─ 📦 加载历史消息
    └─ 🔄 标记为已读

⚡ 搜索频道
├─ ID: channel-search-001
├─ API: GET /communication/channels/search?q={query}
├─ 数据：💾 searchResults
├─ 事件：
│  ├─ 🔌 onInput: searchChannels
│  └─ ⚠️ 防抖 300ms
└─ 备注：
    ├─ 📦 搜索范围：频道名/消息内容
    └─ 🎨 高亮匹配项

🔐 私人频道加密
├─ ID: channel-encrypt-001
├─ 数据：💾 encryptionKey
├─ 事件：
│  ├─ 🔐 建立连接时：generateKey
│  └─ 🔐 消息发送/接收：encrypt/decrypt
└─ 备注：
    ├─ 🔒 端到端加密
    └─ 💾 localStorage 存储密钥
```
