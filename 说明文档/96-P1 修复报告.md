## 📋 问题描述

### bare except 吞没异常

**问题代码模式:**
```python
try:
    # 某些操作
    pass
except:  # ❌ 吞没所有异常，包括 KeyboardInterrupt
    continue  # 或 pass
```

**影响:**
- 🔴 吞没所有异常，包括 `KeyboardInterrupt`、`SystemExit`
- 🔴 调试困难，不知道发生了什么错误
- 🔴 可能导致程序无法正常退出
- 🔴 违反 Python 最佳实践
