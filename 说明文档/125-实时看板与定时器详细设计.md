## 3️⃣ 心跳状态模块（埋点标注）

```markdown
## 💓 心跳状态

📡 WebSocket 连接
├─ ID: heartbeat-ws-001
├─ API: WebSocket /ws/heartbeat
├─ 数据：
│  ├─ 💾 heartbeat.status (✅正常/⚠️警告/❌异常)
│  ├─ 💾 heartbeat.last_check
│  └─ 💾 heartbeat.response_time
├─ 事件：
│  ├─ 📡 onMessage: updateHeartbeat
│  ├─ 📡 onOpen: wsConnected
│  └─ 📡 onClose: wsDisconnected
└─ 备注：
    ├─ 🔌 断线自动重连（每 5 秒）
    └─ ⚠️ 3 次重连失败 → 显示错误

🔄 心跳轮询（备用）
├─ ID: heartbeat-polling-001
├─ API: GET /agent/heartbeat
├─ 数据：💾 heartbeatStatus
├─ 事件：🔄 每 30 秒轮询
└─ 备注：WebSocket 失败时启用

🎨 动态样式
├─ ID: heartbeat-style-001
├─ 样式规则：
│  ├─ status=正常 → 🟢 绿色圆点 + 动画
│  ├─ status=警告 → 🟡 黄色圆点
│  └─ status=异常 → 🔴 红色圆点 + 闪烁
└─ 备注：视觉化健康状态
```
