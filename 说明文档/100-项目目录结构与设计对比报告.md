前端 API 地址通过环境变量配置：

```javascript
// console/src/api/http.js
const BASE_URL = import.meta.env.VITE_API_URL || 'http://127.0.0.1:8089'
```

**支持配置方式：**

1. `.env` 文件：
   ```env
   VITE_API_URL=http://127.0.0.1:8089
   ```

2. 命令行：
   ```bash
   VITE_API_URL=http://192.168.1.100:8089 npm run build
   ```

## 🚀 U 盘便携性评估

### ✅ 支持便携的特性

1. **相对路径导入** - Python 模块使用相对导入
2. **环境变量配置** - 前端 API 地址可配置
3. **启动脚本** - `便携版启动.bat` 自动检测路径
4. **数据本地化** - 所有数据存储在 `.copaw/` 目录
5. **虚拟环境** - `.venv/` 包含所有 Python 依赖

### ⚠️ 需要注意的事项

1. **绝对路径日志** - 日志中可能显示绝对路径（不影响功能）
2. **数据库连接** - SQLite 使用相对路径，支持移动
3. **Node.js 依赖** - `console/node_modules/` 需一起复制

### 📋 便携版使用指南

```bash
# 1. 插入 U 盘，记住盘符（如 E:\）
# 2. 双击运行 "便携版启动.bat"
# 3. 后端自动启动在 http://localhost:8089
# 4. 前端访问 http://localhost:8089/
```
