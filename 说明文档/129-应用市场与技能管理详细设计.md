## 2️⃣ MCP 插件管理（埋点标注）

```markdown
## 🔌 MCP 插件管理

⚡ 获取 MCP 插件列表
├─ ID: mcp-list-get-001
├─ API: GET /marketplace/mcp
├─ 数据：💾 mcpPlugins (数组)
├─ 事件：onMount + 🔄 每 10 秒刷新
└─ 备注：
    ├─ 📦 数据来源：已安装的 MCP Server
    └─ 💾 字段：name, status, health, installed

💾 MCP 插件卡片
├─ ID: mcp-card-001
├─ 数据：
│  ├─ 💾 plugin.name
│  ├─ 💾 plugin.description
│  ├─ 💾 plugin.status (installed/not_installed)
│  ├─ 💾 plugin.health (healthy/unhealthy/unknown)
│  └─ 💾 plugin.running (running/stopped)
├─ 事件：
│  ├─ 🔌 onInstall: installPlugin
│  ├─ 🔌 onConfigure: openConfigModal
│  ├─ 🔌 onRestart: restartPlugin
│  └─ 🔌 onUninstall: uninstallPlugin
└─ 备注：
    ├─ 🎨 installed=true → 显示"已安装"
    ├─ 🎨 running=true → 绿色状态灯
    └─ 🎨 health=error → 红色状态灯

⚡ 安装 MCP 插件
├─ ID: mcp-install-001
├─ API: POST /marketplace/mcp/install
├─ 数据：
│  ├─ 💾 plugin_name
│  ├─ 💾 command (npx 命令)
│  ├─ 💾 args (参数数组)
│  └─ 💾 port (端口号)
├─ 事件：
│  ├─ 🔌 onClick: openInstallModal
│  └─ 🔌 onSubmit: handleInstall
└─ 备注：
    ├─ ⚠️ 验证：命令格式
    ├─ 📦 数据转换：命令 → 数组
    └─ 🎨 安装进度显示

🔄 安装进度监控
├─ ID: mcp-install-progress-001
├─ API: WebSocket /ws/mcp/install/{id}
├─ 数据：
│  ├─ 💾 progress.status (installing/running/error)
│  ├─ 💾 progress.message (日志信息)
│  └─ 💾 progress.percent (0-100)
├─ 事件：
│  ├─ 📡 onMessage: updateProgress
│  └─ 📡 onComplete: closeProgressModal
└─ 备注：
    ├─ 🎨 实时显示安装日志
    └─ ⚠️ 安装失败显示错误

⚡ 注册到后台技能表
├─ ID: mcp-register-001
├─ API: POST /marketplace/mcp/register
├─ 数据：
│  ├─ 💾 plugin_name
│  ├─ 💾 plugin_type (mcp)
│  ├─ 💾 capabilities (能力列表)
│  └─ 💾 metadata (元数据)
├─ 事件：
│  └─ 🔌 onInstallComplete: registerPlugin
└─ 备注：
    ├─ 📦 自动注册（安装成功后）
    ├─ 💾 注册到 skills 表
    └─ 🔌 刷新技能列表

🔌 配置插件
├─ ID: mcp-configure-001
├─ 事件：
│  ├─ 🔌 onClick: openConfigModal
│  └─ ⚡ 保存配置：PUT /marketplace/mcp/{name}/config
├─ 数据：
│  ├─ 💾 plugin_name
│  └─ 💾 config (键值对)
└─ 备注：
    ├─ ⚠️ 验证：配置格式
    └─ 💾 配置持久化

🔄 健康检查
├─ ID: mcp-health-001
├─ API: GET /marketplace/mcp/{name}/health
├─ 数据：💾 healthStatus
├─ 事件：🔄 每 30 秒刷新
└─ 备注：
    ├─ 📡 WebSocket 推送（可选）
    └─ ⚠️ 连续 3 次失败 → 显示警告

⚠️ 错误处理
├─ ID: mcp-error-001
├─ 错误场景：
│  ├─ 安装失败 → ⚠️ 显示错误日志，🔌 重试按钮
│  ├─ 注册失败 → ⚠️ 手动注册提示
│  ├─ 健康检查失败 → 🔄 自动重试 3 次
│  └─ WebSocket 断开 → 🔌 自动重连
└─ 备注：统一错误处理
```
