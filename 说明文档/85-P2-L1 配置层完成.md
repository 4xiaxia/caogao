#### 验证规则 ✅
```python
@field_validator("providers")
@classmethod
def validate_providers(cls, v: ProvidersConfig) -> ProvidersConfig:
    # 检查 ID 唯一性
    ids = [p.id for p in v.providers]
    if len(ids) != len(set(ids)):
        raise ValueError("Provider IDs must be unique")
    return v
```
